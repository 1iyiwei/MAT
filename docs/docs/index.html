number → number
Parameters
  • x
Returns number

Floating-point safer version of acos. If x is larger than 1 (or smaller than -1), still returns 0 (or Math.PI) instead of NAN.

Util.acos(1);  //=> 0
Util.acos(2);  //=> 0
